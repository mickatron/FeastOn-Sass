    get-color

    @function get-color($palette, $color:null, $modify:null, $modify-value:null) { ... }

    Description

    retrieve color from $color-palette map ie. color(base, primary)

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $palette

    The palette map you wish to query for a colour.

    		Map— none
    $color:null

    The name of the colour in the palette map.

    		String— none
    $modify:null

    The type of modification you wish to apply to the colour. possible values are: rgba, light or dark

    		String— none
    $modify-value:null

    The amount of modification you wish to apply to the colour. The data type will depend on the modification you're performing.

    		Percentage or Number— none

    Returns

    Hex or Rgba —

    The colour value requested.

  base: (
    breakpoint: 0,
    width: 100%,

    base-size: map-get($base, font-size),

  ),
  mobile: (
    breakpoint: $mobile-start,

    gutter: 10px,
    base-size: 12px,
    line-height: 1.3,
    ratio: map-get($scales, minorThird),
    header-height: 56px
  ), 
  tablet: (
    breakpoint: $tablet-start,
    //width: 100%,
    gutter: 5%,

    base-line-height: 1.7,
    ratio: map-get($scales, fourth),
    header-height: 100vh
  ),
  desktop: (
    breakpoint: $desktop-start,
    width: 1440px,
    //gutter: 15%,

  desktopLarge: (

    base-size: 20px,
    grid-gutter: 5%,

  ),
);

    Default responsive $responsive-settings as used in responsive mixins such as font-size and responsive-gutters.
